# Prem Astrology Platform

## Overview

Prem Astrology Platform is an AI-powered astrology report generation system designed to automate the creation and delivery of personalized horoscope reports.

The platform combines traditional Vedic astrology principles, horoscope analysis, Python-based rule engines, and artificial intelligence to generate detailed reports for customers.

---

## Features

### Customer Management

* Customer registration
* Birth details collection
* Question submission
* Order tracking

### Astrology Engine

* Rasi chart analysis
* Dasa Bhukti analysis
* Ashtakavarga analysis

* House-wise analysis

* Automated horoscope interpretation
